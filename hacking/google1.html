The Basics :
- We have the " intitle : " query e.g : intitle:"index of" looking for file indexing
- also the "inurl : " query you maybe want to find an url parameter e.g : inurl:"view.shtml"
- "site :" for looking in various websites (pastebin,anonbin...)
and so many more others you can get from GHDB

Weaponized Examples :
- use < site:target.com filetype:pdf > (or xlsx,docx...) you may find some leaked stuff
- use < site:*.*." target.com > for subdomains
- use < site:" target.com " inurl:signup | inurl:register | intitle:Signup > for login/signup pages
and so many others (be creative), as initiale of automating this checkout Grecon
